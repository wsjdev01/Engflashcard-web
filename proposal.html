01 — 서비스 목적

왜 만들었나

배경

통번역사 시험 준비 중, 교재의 영어 문장을 반복 암기할 효율적인 방법이 필요했음. 기존 교재는 이동 중 꺼내보기 불편하고, 단순 반복 읽기는 인출 효과가 낮음.

목적

스마트폰으로 언제 어디서나 접근 가능한 학습 환경 제공. 의미와 설명을 먼저 보고 영어 문장을 유추하는 인출 학습법 적용.

대상 사용자

통번역사 시험을 준비하는 본인. 단일 사용자 기준으로 설계된 개인 학습 도구.

03 — 기술 스택

어떻게 만들었나

📄
HTML / CSS / JS
단일 파일(index.html)로 전체 앱 구성. 별도 프레임워크 없음.
📊
Google Sheets
"웹에 게시" CSV 방식. API 키 없이 데이터 연동. 시트 수정 즉시 반영.
💾
localStorage
브라우저에 진행상황, 북마크, 카드 순서 저장. 서버 불필요.
▲
Vercel
GitHub push 시 자동 배포. 무료 플랜으로 운영 중.
🐙
GitHub
소스코드 버전 관리. Vercel과 자동 연동.
🤖
Claude Code
바이브코딩 도구. 코드 작성·수정·배포를 AI가 주도.
